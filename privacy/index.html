マッピン（Mappin） プライバシーポリシー

制定日: 2026-06-26 / 最終更新日: 2026-07-05

マッピン（Mappin）は、旅の記録を保存し、必要に応じて共有できるようにするため、以下の情報を取得または保存します。

1. 取得・保存する情報

  • Firebase Authentication を通じたユーザー識別子
  • Apple または Google ログイン時のメールアドレスや表示名
  • 場所名、カテゴリ、住所、電話番号、Google Place ID、緯度・経度
  • 写真、コメント、評価、お気に入り状態
  • 共有マップ、参加メンバー、共有招待に関する情報
  • 表示高速化のためのローカルキャッシュ
  • App Storeの商品ID、取引識別子、購入・更新・解約・返金状態
  • Mixpanelによるアプリ利用状況の分析データ（識別子、メールアドレス、画面遷移や操作イベント）

2. 利用目的

  • 旅の記録の作成、保存、表示
  • 地図への表示と場所検索
  • 共有マップ機能の提供
  • ログインと本人識別
  • アカウント削除、ログアウト、共有取消などのユーザー操作への対応
  • 購入の検証、不正利用防止、プレミアム機能と保存容量の提供

3. 外部サービス

  • Firebase Authentication
  • Firebase Firestore
  • Firebase Storage
  • Apple Sign In
  • Google Sign-In
  • Google Places API
  • Apple StoreKit / App Store Server Notifications
  • Apple WeatherKit
  • Mixpanel

場所検索時には検索語や関連情報が Google Places API に送信されます。ログイン時には Apple または Google と Firebase Authentication に認証情報が送信されます。記録データと画像は Firebase Firestore および Firebase Storage に保存されます。アプリ内課金ではAppleが発行した署名付き取引情報をFirebase Functionsで検証します。クレジットカード番号などの決済情報を本アプリが取得・保存することはありません。記録に登録した場所の緯度・経度と訪問日時は、天気情報を表示するためApple WeatherKitへ送信されます。アプリの利用状況を把握するため、識別子・メールアドレス・画面遷移や操作イベントがMixpanelへ送信されます。

4. 情報の共有

共有機能を利用した場合、共有対象のマップに参加しているユーザーに対し、そのマップ内の記録や共有設定が表示されることがあります。

5. 保存期間と削除

  • アプリ内のアカウント削除機能から、自分の保存データ削除を開始できます。
  • アカウント削除時には、Firebase 上の記録データ、画像、共有設定、および端末キャッシュを削除対象とします。
  • Appleの取引記録は、返金対応、不正利用防止および購入復元に必要な範囲で保持します。
  • 法令上保持が必要な情報がある場合を除き、削除要請に応じて速やかに対応します。

6. 安全管理

Firebase セキュリティルールなどを利用し、ユーザー本人のみが自分のデータへアクセスできるよう制御します。

7. お問い合わせ

サポートや個人情報に関するお問い合わせは、サポート窓口からご連絡ください。
